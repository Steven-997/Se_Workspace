[ 1 ] Name 5 IT job roles and write one line about what each role does.
Here are 5 common IT job roles :

1. Software Developer – Designs, codes, and tests applications or systems to solve problems or meet user needs.

2. Network Engineer – Plans, builds, and maintains an organization's computer networks to ensure reliable connectivity.

3. Cybersecurity Analyst – Monitors systems for threats and vulnerabilities, protecting data and infrastructure from cyberattacks.

4. Cloud Architect – Designs and oversees cloud computing strategies, ensuring scalable and secure cloud infrastructure.

5. Data Scientist – Analyzes large datasets using statistics and machine learning to extract insights and support decision-making.

[ 3 ] Write the full forms and one-line use of: GitHub, Slack, Jira, IDE, API.
GitHub - GitHub  
Use - A platform for hosting, sharing, and collaborating on code using Git version control.

Slack - Searchable Log of All Conversation and Knowledge
Use - A messaging and collaboration tool for teams to communicate in organized channels.

Jira - JIRA (derived from Gojira, Japanese for Godzilla)
Use - A project management tool for tracking bugs, tasks, and agile workflows.

IDE - Integrated Development Environment
Use - A software application that provides tools like a code editor, debugger, and compiler in one place.

API - Application Programming Interface
Use- A set of rules that allows different software applications to communicate with each other.

[ 5 ] What is the difference between a developer and a tester? Answer in 2 lines.
A developer writes and builds the code that creates a software application, while a tester checks that code for bugs, errors, and ensures the software works as expected.
